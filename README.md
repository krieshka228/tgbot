# Макс-бот для магазина

Автоматизированный бот для мессенджера **Макс** (max.ru).

## Возможности

- 📦 Считывает товары из постов канала (название + цена + фото)
- 💬 Клиент пишет количество в комментарии → добавляется в корзину, комментарий удаляется
- 🛒 Личная корзина: просмотр, изменение количества, удаление позиций
- 📄 Формирование счёта на оплату с реквизитами
- 📷 Приём фото-чека, уведомление администратора
- ✅ Подтверждение/отклонение оплаты администратором
- 🚚 Запрос адреса доставки после подтверждения
- ⏰ Автоматические напоминания об оплате (день 1 и день 2 в 09:00)
- 📊 Excel: сводная таблица по товарам
- 📋 Excel: детализация по каждому клиенту
- 👥 Excel: база клиентов
- ✉️ Кнопка «Написать администратору»
- 🔒 Согласие на обработку персональных данных

## Установка

```bash
# 1. Клонируйте/скопируйте проект
cd maxbot

# 2. Создайте виртуальное окружение
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Настройте окружение
cp .env .env
# Откройте .env и заполните все значения
```

## Настройка

Скопируйте `.env.example` в `.env` и заполните:

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота из MasterBot (max.ru/masterbot) |
| `ADMIN_USER_ID` | Ваш user_id в Макс |
| `ADMIN_CHAT_ID` | ID чата для уведомлений (обычно = ADMIN_USER_ID) |
| `CHANNEL_ID` | ID канала/группы с постами товаров |
| `DATABASE_URL` | Путь к БД (по умолчанию SQLite) |
| `PAYMENT_DETAILS` | Реквизиты для оплаты |

## Настройка канала

Бот должен быть **администратором** вашего канала/группы с правами:
- Удаление сообщений (для автоудаления комментариев)
- Просмотр сообщений

## Запуск

```bash
python main.py
```

## Формат постов с товарами

Бот автоматически парсит посты. Пост должен содержать:
- **Первая строка** — название товара
- **Строка с ценой** — в формате: `Цена: 1500 ₽` или `Цена: 1500`

Пример:
```
Диван угловой Milano 2024
Артикул: DV-001
Цена: 45000 ₽
Цвет: серый, бежевый
```

## Как делают заказ клиенты

1. Клиент видит пост с товаром в вашем канале
2. Пишет количество в комментарии: `2` или `2 шт`
3. Бот удаляет комментарий и отправляет ЛС с подтверждением
4. Клиент заходит в бот → видит корзину → оформляет заказ
5. Получает реквизиты → оплачивает → присылает чек
6. Вы подтверждаете → клиент вводит адрес доставки

## Деплой на сервер (Яндекс Облако / VPS)

```bash
# Установка как systemd сервис
sudo nano /etc/systemd/system/maxbot.service
```

```ini
[Unit]
Description=Max Shop Bot
After=network.target

[Service]
WorkingDirectory=/path/to/maxbot
ExecStart=/path/to/maxbot/venv/bin/python main.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl enable maxbot
sudo systemctl start maxbot
sudo systemctl status maxbot
```
